# Simulator de Ciclos de Lavadora 🧺

Este proyecto es un simulador lógico desarrollado en Python que emula el funcionamiento de una lavadora industrial/doméstica. Fue diseñado originalmente como un proyecto académico para la materia de Algoritmos y Estructura de Datos, enfocándose en la gestión de estados y flujos de control.

## 🚀 Características

- **Interfaz Gráfica/Consola:** (Elige la que aplique) Interfaz intuitiva para interactuar con el usuario.
- **Ciclos Personalizados:** Selección de diferentes tipos de lavado (Rápido, Delicado, Pesado).
- **Control de Estados:** Gestión precisa de las etapas: Llenado, Lavado, Enjuague y Centrifugado.
- **Validación de Parámetros:** Control de peso de carga y niveles de agua para optimizar recursos.

## 🛠️ Tecnologías Utilizadas

* **Lenguaje:** Python3, Tkinter
* **Lógica:** Estructuras de control de flujo y programación orientada a objetos (POO).
* **Gestión de Dependencias:** Virtualenv (configurado localmente).
